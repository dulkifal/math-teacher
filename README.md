## ✨ Key Features & Interactive Lessons

The application offers a suite of interactive modules designed to visualizes complex math concepts:

*   **📐 Interactive Angles:** Manipulate angles dynamically to understand geometric properties in real-time.
*   **🪞 Symmetry Visualizer:** Drag and rotate shapes to observe reflectional and rotational symmetry.
*   **🔺 Triangle Inspector:** Interact with vertices and side lengths to explore geometric theorems dynamically.
*   **➕ Gesture-Based Addition:** Touch and slide objects to intuitively grasp basic arithmetic.
*   **🧊 3D/2D Cubes & Spatial Reasoning:** Interact with dimensions to conceptualize 3D geometry.
*   **🔐 Secure User Journeys:** Full authentication system with role-based features powered by Clerk.

## 🛠️ The Tech Stack & Engineering Standards

*   **Frontend Framework:** Next.js (React 19) utilizing TypeScript for compile-time safety and self-documenting code.
*   **Styling & Design System:** Tailwind CSS v4 for modern, lightning-fast styling, fluid transitions, and responsive layouts.
*   **Gestures & Physics-based Animations:** Powered by `react-spring` and `react-use-gesture` to create smooth, natural interactions that respond instantly to user input.
*   **Data Persistence & Modeling:** Prisma ORM connecting to a relational database, ensuring robust data relationships and migrations.
*   **Security:** Integrated Clerk Auth for OAuth logins, session management, and secure API routes.

## ⚙️ Development Setup

If you want to run this project locally to inspect the codebase quality:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/math-teacher.git
   cd math-teacher
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure environment variables:**
   Create a `.env` file in the root directory based on `.env.example` and supply your database URL and Clerk credentials.

4. **Run database migrations:**
   ```bash
   npx prisma db push
   ```

5. **Start the development server:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) to see it in action.
